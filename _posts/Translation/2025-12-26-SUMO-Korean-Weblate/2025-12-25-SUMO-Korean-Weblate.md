## Eclipse SUMO 한국어 번역 페이지 개설

2025년 12월 25일, [Eclipse SUMO](https://eclipse.dev/sumo/)의 Weblate 번역 페이지에 한국어가 추가되었습니다.

SUMO(Simulation of Urban Mobility)는 도시 교통 시뮬레이션을 위한 오픈소스 소프트웨어로, 대규모 도로 네트워크에서의 차량, 보행자 등 다양한 교통 수단의 시뮬레이션을 지원합니다.
교통 연구, 자율주행 개발, 도시 계획 등 다양한 분야에서 활용되고 있으며, 특히 학생 연구자들에게 많이 활용되는 도구인 만큼 한국어 지원이 큰 도움이 될 수 있을 거라 생각합니다.
<br>

### 배경

Eclipse SUMO의 GitHub 저장소에 한국어 번역 추가를 요청하는 Issue를 등록했고, eclipse-sumo 측에서 Weblate에 티켓을 생성하여 한국어 번역 페이지를 추가해주었습니다.

현재 번역을 진행하고 있지만, 저는 SUMO의 일부 기능과 라이브러리 위주로 사용해왔기 때문에 전체 기능에 대한 번역에 어려움이 있습니다.
SUMO는 GUI, Netedit, CLI 애플리케이션, Python 도구 등 컴포넌트가 다양하고, 각 분야에 대한 이해가 필요한 부분이 많습니다.
<br>

### 참여를 부탁드리는 이유

사실 대부분의 연구자분들은 영어를 읽는 데 큰 어려움이 없으실 겁니다.
하지만, 아직 영어가 서툰 학부생, 아마추어, 주니어 연구자 등 후배 연구자분들에게는 한국어 지원이 진입 장벽을 낮추는 데 큰 도움이 될 수 있습니다.

SUMO를 사용해보신 분들이라면, 본인이 잘 아는 기능의 번역 몇 줄만으로도 큰 기여가 됩니다.
많은 참여 부탁드립니다.
<br>

## 번역 참여 방법

Weblate를 통해 누구나 웹 브라우저에서 번역에 참여할 수 있습니다.

1. [Eclipse SUMO Weblate 페이지](https://hosted.weblate.org/projects/eclipse-sumo/)에 접속합니다.
2. Weblate 계정을 생성하거나, GitHub 계정으로 로그인합니다.
3. 아래 컴포넌트 중 원하는 항목의 한국어(Korean) 페이지에서 번역을 시작합니다.

### 번역 컴포넌트

| 컴포넌트 | 설명 | 번역 페이지 |
|---------|------|-----------|
| SUMO GUI / Netedit | GUI 및 네트워크 편집기 | [Korean](https://hosted.weblate.org/projects/eclipse-sumo/sumo-gui-netedit/ko/) |
| Applications | CLI 애플리케이션 | [Korean](https://hosted.weblate.org/projects/eclipse-sumo/applications/ko/) |
| Python tools | Python 도구 | [Korean](https://hosted.weblate.org/projects/eclipse-sumo/python-tools/ko/) |
| Glossary | 용어집 | [Korean](https://hosted.weblate.org/projects/eclipse-sumo/glossary/ko/) |
